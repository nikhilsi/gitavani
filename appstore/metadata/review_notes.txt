GitaVani is an offline Bhagavad Gita reader. All 701 verses are bundled locally — the app makes no network calls. Interactive features include: swipe gesture and button navigation between verses (including cross-chapter), Hindi/English language toggle with 12-author picker, Sanskrit transliteration toggle, commentaries from 17 scholars in 3 languages (English/Hindi/Sanskrit) with author picker and Read more for long texts, pull-down search on home screen and chapter detail, favorites list with sort toggle, share verse as themed image card, 4 color themes, adjustable font size (14-28pt) with Dynamic Type support, persistent reading progress with resume banner, onboarding flow, and help screen.
